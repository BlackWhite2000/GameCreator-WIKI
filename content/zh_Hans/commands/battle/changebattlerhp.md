## 指定方式

该指令可以通过多种方式去指定角色，合理使用可以制作出意想不到的效果。请一定要善用指定方式，这是一个非常好用智能的功能。

| 指定方式             | 说明                             |
| -------------------- | -------------------------------- |
| 所在玩家队伍中的编号 | 战斗时，通过玩家队伍的位置来指定 |
| 所在敌方队伍中的编号 | 战斗时，通过敌人的队伍位置来指定 |
| 当前战斗的行动者     | 战斗时，当前正在行动的角色       |
| 当前战斗的被击中者   | 战斗时，当前被攻击击中的角色     |
| 全体玩家角色         | 战斗时，玩家的全部角色           |
| 全体敌方角色         | 战斗时，敌方的全部角色           |

## 使用

选择好指定方式以及角色，即可在战斗中增加、减少该角色的生命值。

例: 增加 `敌方队伍编号 1` 的角色生命值 `200`
