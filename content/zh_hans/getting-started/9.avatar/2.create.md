## 进入行走图编辑器

1-打开数据库，点击数据库图标或快捷键F10打开数据库

![alt text](/image/zh_hans/getting-started/9.avatar/2.create/image.png)

2-切换至行走图编辑器

![alt text](/image/zh_hans/getting-started/9.avatar/2.create/image-1.png)

## 数据列表

行走图数据统一采用编号进行管理。

分为16个类别，可以在每个分类中更改最大值，以便调整可用的行走图范围。

![alt text](/image/zh_hans/getting-started/9.avatar/2.create/image-2.png)

![alt text](/image/zh_hans/getting-started/9.avatar/2.create/image-3.png)

![alt text](/image/zh_hans/getting-started/9.avatar/2.create/image-4.png)

## 选择行走图数据

鼠标左键单击指定的行走图可打开该行走图数据进行编辑。

## 其他操作

### 复制

复制一个行走图数据

| 操作方式 |                    |
| -------- | ------------------ |
| 键盘操作 | `CTRL+C`           |
| 鼠标操作 | `鼠标右键选择复制` |

### 粘贴

将此前复制的行走图数据粘贴到当前指定的数据位置上

| 操作方式 |                    |
| -------- | ------------------ |
| 键盘操作 | `CTRL+V`           |
| 鼠标操作 | `鼠标右键选择粘贴` |

### 清空

清空当前选择的数据，由于GameCreator数据由编号统一管理，

数据只能清空和更改最大值，无法删除。

| 操作方式 |                    |
| -------- | ------------------ |
| 键盘操作 | `CTRL+D`           |
| 鼠标操作 | `鼠标右键选择清空` |

## 重命名分类

可以给分类进行命名

| 操作方式 |                                        |
| -------- | -------------------------------------- |
| 鼠标操作 | `鼠标右键点击分类弹出菜单，选择重命名` |
