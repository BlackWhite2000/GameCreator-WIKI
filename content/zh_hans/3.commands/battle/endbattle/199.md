## 使用

战斗中使用即可，支持多种结束方式。

例: 战斗结束，战斗结果为胜利

![alt text](https://cdn.gcw.wiki/gcw/image/zh_hans/commands/battle/endbattle/199/image.png)

| 结束方式 | 说明                         |
| -------- | ---------------------------- |
| 默认     | 系统使用，作者不需要使用到   |
| 胜利     | 战斗结束，战斗结果为玩家胜利 |
| 失败     | 战斗结束，战斗结果为玩家失败 |
| 逃跑     | 战斗结束，战斗结果为玩家逃跑 |
