## 说明

游戏服务器的一些常用参数配置

## Public 属性

|                                                                               |
| ----------------------------------------------------------------------------- |
| **SCENE_FIXED_THREAD_COUNT** : number;<br>【静态】开启的固定场景线程数（3~N） |
| **MAX_CONN** : number;<br>【静态】允许登陆的服务器上线人数                    |
| **MYSQL_CONN_HOST** : string;<br>【静态】数据库服务器地址                     |
| **MYSQL_CONN_PORT** : string;<br>【静态】数据库服务器端口                     |
| **MYSQL_CONN_USERNAME** : string;<br>【静态】数据库服务器用户名               |
| **MYSQL_CONN_PASSWORD** : string;<br>【静态】数据库服务器密码                 |
| **MYSQL_CONN_DATABASE** : string;<br>【静态】数据库名称                       |
