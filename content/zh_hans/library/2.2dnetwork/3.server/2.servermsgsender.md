- `继承` 无
- `子类` 无

## 说明

用于将消息发送给客户端，同时封装了直接调用客户端方法（RPC）的功能

## Public 方法

|                                                                                                                                        |
| -------------------------------------------------------------------------------------------------------------------------------------- |
| **[send](#send)**(player : ServerPlayer,  msg : string): void<br>【静态】发送字符串消息                                                |
| **[eval](#eval)**(player : ServerPlayer,  code : string): void<br>【静态】让客户端执行代码                                             |
| **[rpc](#rpc)**(player : ServerPlayer,  className : string,  funcName : string,  params : any[]): void<br>【静态】远程调用客户端的方法 |

## 详情

### send

```ts
send(player : ServerPlayer,  msg : string): void :
```

【静态】发送字符串消息

- #### 参数

  - `player` 玩家
  - `msg` 字符串消息

### eval

```ts
eval(player : ServerPlayer,  code : string): void :
```

【静态】让客户端执行代码

- #### 参数

  - `player` 玩家
  - `code` 客户端代码

### rpc

```ts
rpc(player : ServerPlayer,  className : string,  funcName : string,  params : any[]): void :
```

【静态】远程调用客户端的方法

- #### 参数

  - `className` 类名
  - `funcName` 方法名
  - `params` 参数
