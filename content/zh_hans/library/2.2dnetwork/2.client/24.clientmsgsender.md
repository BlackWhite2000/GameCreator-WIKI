## 详情

### send

```ts
send(msg : string,  threadID? : number): void :
```

【静态】发送字符串消息

- #### 参数

  - `msg`
  - `threadID` [可选] 默认值=-1 指定发送的线程，未指定则说明是当前玩家所在的场景线程 2=主线程 3~N=场景线程 -1=表示默认玩家所在的线程

### rpc

```ts
rpc(className : string,  funcName : string,  params? : any[],  onReturn? : Callback,  threadID? : number): void :
```

【静态】远程调用服务器的方法（需要服务器设置调用权限 [ServerWorld]的addServerFunction）<br>

- 在玩家进入场景后才能使用该方法

- #### 参数

  - `className` 调用的类
  - `funcName` 调用的方法
  - `params` 方法参数
  - `onReturn` 服务器返回值回调，超时会被删除
  - `threadID` 指定发送的线程，未指定则说明是当前玩家所在的场景线程 2=主线程 3~N=场景线程 -1=表示默认玩家所在的线程

### requestListenPlayerVariable

```ts
requestListenPlayerVariable(isListen : boolean,  type : number,  varID : number): void :
```

【静态】请求监听玩家变量

- #### 参数

  - `isListen` 是否监听
  - `type` 类别 0-变量 1-开关 2-字符串
  - `varID` 变量ID

### requestGetWorldVariable

```ts
requestGetWorldVariable(type : number,  varID : number): void :
```

【静态】请求获取世界变量

- #### 参数

  - `type` 0-变量 1-开关 2-字符串
  - `varID` 变量ID
