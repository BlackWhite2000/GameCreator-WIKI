## 说明

2D变换：表示一个转换矩阵，它确定如何将点从一个坐标空间映射到另一个坐标空间。您可以对一个显示对象执行不同的图形转换，方法是设置 Matrix 对象的属性，

将该 Matrix 对象应用于 Transform 对象的 matrix 属性，然后应用该 Transform 对象作为显示对象的 transform 属性。这些转换函数包括平移（x 和 y 重新定位）、旋转、缩放和倾斜

> a c tx<br>
> b d ty<br>
> 平移：tx、ty<br>
> 缩放：a(scaleX)、d(scaleY)<br>
> 旋转: q表示弧度<br>
> cos(q) -sin(q) tx<br>
> sin(q) cos(q) ty

## 详情

### EMPTY

```ts [签名]
EMPTY : Matrix;
```

【静态】空矩阵：

1 0 0

0 1 0

### constructor

```ts [签名]
constructor(a? : number, b? : number, c? : number, d? : number, tx? : number, ty? : number)
```

构造函数

- #### 参数

  - `a` [可选] 默认值=1 缩放或旋转图像时影响像素沿 x 轴定位的值。
  - `b` [可选] 默认值=0 旋转或倾斜图像时影响像素沿 y 轴定位的值。
  - `c` [可选] 默认值=0 旋转或倾斜图像时影响像素沿 x 轴定位的值。
  - `d` [可选] 默认值=1 缩放或旋转图像时影响像素沿 y 轴定位的值。
  - `tx` [可选] 默认值=0 沿 x 轴平移每个点的距离。
  - `ty` [可选] 默认值=0 沿 y 轴平移每个点的距离。

### identity

```ts [签名]
identity(): Matrix
```

将本矩阵设置为单位矩阵。

- #### 返回

  - `Matrix` 返回当前矩形。

### setTranslate

```ts [签名]
setTranslate(x : number, y : number): Matrix
```

设置位置：直接设置tx、ty。

- #### 参数

  - `x` 沿 x 轴平移每个点的距离。
  - `y` 沿 y 轴平移每个点的距离。

- #### 返回

  - `Matrix` 返回对象本身

### translate

```ts [签名]
translate(x : number, y : number): Matrix
```

平移位置：在原来的tx、ty基础上增加值。

- #### 参数

  - `x` 沿 x 轴向右移动的量（以像素为单位）。
  - `y` 沿 y 轴向下移动的量（以像素为单位）。

- #### 返回

  - `Matrix` 返回此矩形对象。

### scale

```ts [签名]
scale(x : number, y : number): void
```

对矩阵应用缩放转换。

- #### 返回

  - `x` 用于沿 x 轴缩放对象的乘数。1=100%
  - `y` 用于沿 y 轴缩放对象的乘数。1=100%

### rotate

```ts [签名]
rotate(angle : number): void
```

对矩阵应用旋转。

- #### 参数

  - `angle` 以弧度为单位的旋转角度。

### skew

```ts [签名]
skew(x : number, y : number): Matrix
```

对矩阵应用倾斜转换。

- #### 参数

  - `x` 沿着 X 轴的 2D 倾斜弧度。
  - `y` 沿着 Y 轴的 2D 倾斜弧度。

- #### 返回

  - `Matrix` 当前 Matrix 对象。

### transformPoint

```ts [签名]
transformPoint(out : Point): Point
```

将矩阵对象表示的几何转换应用于指定点。

- #### 参数

  - `out` 用来设定输出结果的点。

- #### 返回

  - `Point` 返回out

### invert

```ts [签名]
invert(): Matrix
```

执行原始矩阵的逆转换。

- #### 返回

  - `Matrix` 当前矩阵对象。

### setTo

```ts [签名]
setTo(a : number, b : number, c : number, d : number, tx : number, ty : number): Matrix
```

将矩阵对象的成员设置为指定值。

- #### 参数

  - `a` 缩放或旋转图像时影响像素沿 x 轴定位的值。
  - `b` 旋转或倾斜图像时影响像素沿 y 轴定位的值。
  - `c` 旋转或倾斜图像时影响像素沿 x 轴定位的值。
  - `d` 缩放或旋转图像时影响像素沿 y 轴定位的值。
  - `tx` 沿 x 轴平移每个点的距离。
  - `ty` 沿 y 轴平移每个点的距离。

- #### 返回

  - `Matrix` 当前矩阵。

### concat

```ts [签名]
concat(matrix : Matrix): Matrix
```

将指定矩阵与当前矩阵连接，从而将这两个矩阵的几何效果有效地结合在一起。

- #### 参数

  - `Matrix` 要连接到源矩阵的矩阵。

- #### 返回

  - `Matrix` 当前矩阵。

### clone

```ts [签名]
clone(): Matrix
```

返回此 Matrix 对象的副本。

- #### 返回

  - `Matrix` 与原始实例具有完全相同的属性的新 Matrix 实例。

### toString

```ts [签名]
toString(): string
```

返回列出该 Matrix 对象属性的文本值。

- #### 返回

  - `string` 一个字符串，它包含 Matrix 对象的属性值：a、b、c、d、tx 和 ty。

### destroy

```ts [签名]
destroy(): void
```

销毁此对象。
