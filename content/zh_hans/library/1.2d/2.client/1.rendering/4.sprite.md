## 说明

显示对象的基类，支持显示图片和事件响应

支持通用鼠标事件：

```ts
// 鼠标右键弹起
EventObject.RIGHT_MOUSE_UP 
// 鼠标右键按下
EventObject.RIGHT_MOUSE_DOWN 
// 鼠标右键点击
EventObject.RIGHT_CLICK 
// 鼠标左键点击
EventObject.CLICK 
// 鼠标左键双击
EventObject.DOUBLE_CLICK 
// 鼠标左键按下
EventObject.MOUSE_DOWN 
// 鼠标左键弹起
EventObject.MOUSE_UP 
// 鼠标滚轮
EventObject.MOUSE_WHEEL 
// 鼠标拖拽开始
EventObject.DRAG_START 
// 鼠标拖拽移动中
EventObject.DRAG_MOVE 
// 鼠标拖拽移动结束
EventObject.DRAG_END 
```

```ts
// 事件监听示例
var sp = new Sprite();
sp.on(EventObject.CLICK,this,this.onClick);
```

## 详情

### stage

```ts [签名]
stage : Stage;
```

【只读】若该对象在舞台上则返回舞台，否则返回null

- #### 返回

  - `Stage`

### hitArea

```ts [签名]
hitArea : any;
```

可以设置一个Rectangle区域作为点击区域，设置后则以该区域作为鼠标事件检测

支持类型：HitArea | Rectangle

### mask

```ts [签名]
mask : Sprite;
```

遮罩，可以设置一个对象(支持位图和矢量图)，根据对象形状进行遮罩显示，支持像素级遮罩

遮罩对象坐标系是相对遮罩对象本身的，即以该对象的0,0点为准

### mouseEnabled

```ts [签名]
mouseEnabled : boolean;
```

是否接受鼠标事件

默认为false，如果监听鼠标事件，则会自动设置本对象及父节点的属性 mouseEnable 的值都为 true（如果父节点手动设置为false，则不会更改）

### dispose

```ts [签名]
dispose(): void
```

销毁

### getBounds

```ts [签名]
getBounds(): Rectangle
```

获取本对象在父容器坐标系的矩形显示区域，计算量较大，尽量少用

- #### 返回

  - `Rectangle` 矩形区域

### getSelfBounds

```ts [签名]
getSelfBounds(): Rectangle
```

获取本对象在自己坐标系的矩形显示区域，计算量较大，尽量少用

- #### 返回

  - `Rectangle` 矩形区域

### localToGlobal

```ts [签名]
localToGlobal(point : Point): Point
```

把本地坐标转换为全局坐标

- #### 参数

  - `Point` 本地坐标点

- #### 返回

  - `Point` 转换后的全局坐标

### globalToLocal

```ts [签名]
globalToLocal(point : Point): Point
```

把全局坐标转换为本地坐标

- #### 参数

  - `point` 全局坐标点

- #### 返回

  - `Point` 转换后的坐标的点

### startDrag

```ts [签名]
startDrag(area? : Rectangle): void
```

开始拖动此对象

- #### 参数

  - `area` [可选] 默认值=null 拖动限定的区域范围内

### stopDrag

```ts [签名]
stopDrag(): void
```

停止拖动此对象

### hitTestPoint

```ts [签名]
hitTestPoint(x : number, y : number): boolean
```

检测某个点是否在此对象内

- #### 参数

  - `x` 全局x坐标
  - `y` 全局y坐标。

- #### 返回

  - `boolean` 表示是否在对象内

### pos

```ts [签名]
pos(x : number, y : number, speedMode? : boolean): Sprite
```

设置坐标位置。相当于分别设置x和y属性。

- #### 参数

  - `x` X轴坐标。
  - `y` Y轴坐标。
  - `speedMode` （可选）是否极速模式，正常是调用this.x=value进行赋值，极速模式直接调用内部函数处理，如果未重写x，y属性，建议设置为急速模式性能更高。

- #### 返回

  - `Sprite` 返回对象本身。
