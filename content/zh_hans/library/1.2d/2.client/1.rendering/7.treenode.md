## 说明

显示对象的基类，拥有父子节点的特性，树结构

支持事件：

```ts
// 加入到显示列表中时（即该对象加入stage的显示列表里，从不需要被渲染到需要渲染的状态改变时触发）
EventObject.DISPLAY 
// 从显示列表中移除时（即该对象从stage的显示列表移除时，从需要被渲染到不需要渲染的状态改变时触发）
EventObject.UNDISPLAY 
// 从父节点中移除时触发（无需在显示列表中）
EventObject.REMOVED 
// 加入到某个节点时触发（无需在显示列表中）
EventObject.ADDED 
```

## 详情

### dispose

**[dispose](#dispose)**(): void :

销毁此对象。destroy对象默认会把自己从父节点移除，并且清理自身引用关系，等待js自动垃圾回收机制回收。destroy后不能再使用。

destroy时会移除自身的事情监听，自身的timer监听，移除子对象及从父节点移除自己。

### addChild

```ts
addChild(node : TreeNode): TreeNode :
```

添加子节点。

- #### 参数

  - `node` 节点对象

- #### 返回

  - `TreeNode` 返回添加的节点

### addChildren

```ts
addChildren(...args : any[]): void :
```

批量增加子节点

- #### 参数

  - `...args` 无数子节点。

### addChildAt

```ts
addChildAt(node : TreeNode,  index : number): TreeNode :
```

添加子节点到指定的索引位置。

- #### 参数

  - `node` 节点对象。
  - `index` 索引位置。

- #### 返回

  - `TreeNode` 返回添加的节点。

### getChildIndex

```ts
getChildIndex(node : TreeNode): number :
```

根据子节点对象，获取子节点的索引位置。

- #### 参数

  - `node` 子节点。

- #### 返回

  - `number` 子节点所在的索引位置。

### getChildByName

```ts
getChildByName(name : string): TreeNode :
```

根据子节点的名字，获取子节点对象。

- #### 参数

  - `name` 子节点的名字。

- #### 返回

  - `TreeNode` 节点对象。

### getChildAt

```ts
getChildAt(index : number): TreeNode :
```

根据子节点的索引位置，获取子节点对象。

- #### 参数

  - `index` 索引位置

- #### 返回

  - `TreeNode` 子节点

### setChildIndex

```ts
setChildIndex(node : TreeNode,  index : number): TreeNode :
```

设置子节点的索引位置。

- #### 参数

  - `node` 子节点。
  - `index` 新的索引。

- #### 返回

  - `TreeNode` 返回子节点本身。

### removeChild

```ts
removeChild(node : TreeNode): TreeNode :
```

删除子节点。

- #### 参数

  - `node` 子节点

- #### 返回

  - `TreeNode` 被删除的节点

### removeSelf

**[removeSelf](#removeSelf)**(): TreeNode :

从父容器删除自己，如已经被删除不会抛出异常。

###### 返回

当前节点（ TreeNode ）对象。

### removeChildByName

```ts
removeChildByName(name : string): TreeNode :
```

根据子节点名字删除对应的子节点对象，如果找不到不会抛出异常。

- #### 参数

  - `name` 对象名字。

- #### 返回

  - `TreeNode` 查找到的节点（ TreeNode ）对象。

### removeChildAt

```ts
removeChildAt(index : number): TreeNode :
```

根据子节点索引位置，删除对应的子节点对象。

- #### 参数

  - `index` 节点索引位置。

- #### 返回

  - `TreeNode` 被删除的节点。

### removeChildren

```ts
removeChildren(beginIndex? : number,  endIndex? : number): TreeNode :
```

删除指定索引区间的所有子对象。

- #### 参数

  - `beginIndex` 开始索引。默认值=0
  - `endIndex` 结束索引。默认值=0x7fffffff

- #### 返回

  - `TreeNode` 当前节点对象。

### contains

```ts
contains(node : TreeNode): boolean :
```

当前容器是否包含指定的 TreeNode 节点对象 。

- #### 参数

  - `node` 指定的 TreeNode 节点对象 。

- #### 返回

  - `boolean` 一个布尔值表示是否包含指定的 TreeNode 节点对象 。
