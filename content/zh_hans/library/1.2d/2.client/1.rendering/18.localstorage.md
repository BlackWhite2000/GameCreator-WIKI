## 说明

用于没有时间限制的数据存储,类似网页的永久cookies，清空缓存或更换浏览器都会失效

储存和读取都是同步的，所以可以代码上下行操作

使用方式

```ts
var a = {b:123,c:456,d:"gamecreator"};
// 缓存
LocalStorage.setJSON("myKey",a);
// 读取缓存
LocalStorage.getJSON("myKey",b);
```

## 详情

## setItem

```ts
setItem(key : string,  value : string): void :
```

【静态】存储指定键名和键值，字符串类型。

- #### 参数

  - `key` 键名。
  - `value` 键值。

## getItem

```ts
getItem(key : string): string :
```

【静态】获取指定键名的值。

- #### 参数

  - `key` 键名。

- #### 返回

  - `string` 字符串型值。

## setJSON

```ts
setJSON(key : string,  value : any): void :
```

【静态】存储指定键名及其对应的 Object 类型值。

- #### 参数

  - `key` 键名。
  - `value` 键值。是 Object 类型，此致会被转化为 JSON 字符串存储。

## getJSON

```ts
getJSON(key : string): any :
```

【静态】获取指定键名对应的Object 类型值。

- #### 参数

  - `key` 键名。

- #### 返回

  - `Object` 类型值

## removeItem

```ts
removeItem(key : string): void :
```

【静态】删除指定键名的信息。

- #### 参数

clear(): void :
