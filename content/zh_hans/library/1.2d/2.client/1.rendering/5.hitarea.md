## 关联性

- `继承` 无
- `子类` 无

## 说明

可以设置绘制一系列矢量图作为点击区域和非点击区域（目前只支持圆形，矩形，多边形）

## Public 属性

|                                                                                               |
| --------------------------------------------------------------------------------------------- |
| hit : Graphics;<br>可点击区域，可以设置绘制一系列矢量图作为点击区域（支持圆形，矩形，多边形） |

## Public 方法

|                                                                                                                   |
| ----------------------------------------------------------------------------------------------------------------- |
| [isHitGraphic](#ishitgraphic)(x : number, y : number, graphic : Graphics): boolean<br>【静态】是否击中Graphic     |
| [ptInPolygon](#ptinpolygon)(x : number, y : number, areaPoints : number[]): boolean<br>【静态】坐标是否在多边形内 |
| [isHit](#ishit)(x : number, y : number): boolean<br>是否包含某个点                                                |
| [contains](#contains)(x : number, y : number): boolean<br>检测对象是否包含指定的点。                              |
| (): unHit Graphics<br>不可点击区域，可以设置绘制一系列矢量图作为非点击区域（支持圆形，矩形，多边形）              |

## 详情

### isHitGraphic

```ts [签名]
isHitGraphic(x : number, y : number, graphic : Graphics): boolean :
```

【静态】是否击中Graphic

- #### 参数

  - `x` x坐标
  - `y` y坐标
  - `graphic` 图形源

- #### 返回

  - `boolean`

### ptInPolygon

```ts [签名]
ptInPolygon(x : number, y : number, areaPoints : number[]): boolean :
```

【静态】坐标是否在多边形内

- #### 参数

  - `x` x坐标
  - `y` y坐标
  - `areaPoints` 格式 \[x1,y1,x2,y2...\]

- #### 返回

  - `boolean`

### isHit

```ts [签名]
isHit(x : number, y : number): boolean :
```

是否包含某个点

- #### 参数

  - `x` x坐标
  - `y` y坐标

- #### 返回

  - `boolean` 是否点击到

### contains

```ts [签名]
contains(x : number, y : number): boolean :
```

检测对象是否包含指定的点。

- #### 参数

  - `x` 点的 X 轴坐标值（水平位置）。
  - `y` 点的 Y 轴坐标值（垂直位置）。

- #### 返回

  - `boolean` 如果包含指定的点，则值为 true；否则为 false。
