## 详情

### networkState

```ts
networkState : number;
```

【只读】网络状态

- 0 = NETWORK_EMPTY - 音频/视频尚未初始化
- 1 = NETWORK_IDLE - 音频/视频是活动的且已选取资源，但并未使用网络
- 2 = NETWORK_LOADING - 浏览器正在下载数据
- 3 = NETWORK_NO_SOURCE - 未找到音频/视频来源

### onLoadedFragEvent

```ts
onLoadedFragEvent : string;
```

片段事件内容：当视频源加载完毕时

主动调用方式：CommandPage.startTriggerFragmentEvent
