## 说明

显示文本的组件，支持绑定玩家字符串变量

### 相关事件

```ts
EventObject.CHANGE 文本改变时
```

变量系统在显示时会自动同步显示字符串变量

### 使用方法

```ts
var a = new UIString();
a.text = "kds"; // 固定的文本
a.text = "$6"; // 绑定6号玩家字符串变量
stage.addChild(a);
// 事件监听示例
a.on(EventObject.CHANGE,this,this.onChange);
```
