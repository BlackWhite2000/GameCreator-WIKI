## 关联性

- `继承` 无
- `子类` 无

## 说明

UIListItemData 界面组件中的项数据基类

使用方式可参考 [UIList](./uilist)

## Public 属性

|                                                                               |
| ----------------------------------------------------------------------------- |
| **data** : any;<br>任意附加的数据                                             |
| **isOpen** : boolean;<br>是否处于打开状态（树节点的情况）                     |
| **parent** : UIListItemData;<br>【只读】获取父节点                            |
| **numChildren** : number;<br>【只读】子节点总数                               |
| **children** : UIListItemData[];<br>【只读】子节点列表                        |
| **[root](#root)** : UIListItemData;<br>【只读】获取树形结构的根节点           |
| **[depth](#depth)** : number;<br>【只读】获取所在树的深度                     |
| **isHideNode** : boolean;<br>【只读】是否是隐藏节点（即父系节点可能被关闭了） |

## Public 方法

|                                                                                                                          |
| ------------------------------------------------------------------------------------------------------------------------ |
| **[getSaveData](#getsavedata)**(includeData? : boolean): any<br>提取储存数据                                             |
| **[recoverySaveData](#recoverysavedata)**(saveData : any): UIListItemData<br>【静态】还原储存数据                        |
| **[addChild](#addchild)**(item : UIListItemData): void<br>添加节点                                                       |
| **[addChildAt](#addchildat)**(item : UIListItemData,  index : number): void<br>添加节点到指定索引中                      |
| **[removeChild](#removechild)**(item : UIListItemData): void<br>移除节点                                                 |
| **[removeChildAt](#removechildat)**(index : number): void<br>移除节点至指定索引中                                        |
| **[removeAll()](#removeall)**: void<br>移除所有节点                                                                      |
| **[getChildAt](#getchildat)**(index : number): UIListItemData<br>获取节点根据索引                                        |
| **[getChildIndex](#getchildindex)**(item : UIListItemData): number<br>获取节点索引                                       |
| **[isInherit](#isinherit)**(data : UIListItemData): boolean<br>是否继承于指定节点                                        |
| **[getList](#getlist)**(arr? : UIListItemData[]): UIListItemData[]<br>获取树型结构下全部节点中的子节点列表（含自身节点） |

## 详情

### root

```ts
root : UIListItemData;
```

【只读】获取树形结构的根节点

- #### 返回

  - `UIListItemData`

### depth

```ts
depth : number;
```

【只读】获取所在树的深度

- #### 返回

  - `number`

### getSaveData

```ts
getSaveData(includeData? : boolean): any :
```

提取储存数据

- #### 参数

  - `includeData` [可选] 默认值=false 是否包含自定义附加的数据，请保证该数据可以JSON化

### recoverySaveData

```ts
recoverySaveData(saveData : any): UIListItemData :
```

【静态】还原储存数据

- #### 参数

  - `saveData`

### addChildAt

- #### 参数

### removeChild

```ts
removeChild(item : UIListItemData): void :

removeAll(): void :

## isInherit

isInherit(data : UIListItemData): boolean :
```

## getList

```ts
getList(arr? : UIListItemData[]): UIListItemData[] :
```
