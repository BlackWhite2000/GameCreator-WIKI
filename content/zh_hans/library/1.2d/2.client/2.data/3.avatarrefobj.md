- `继承` 无

## 说明

一般用于自定义Avatar的一些点、范围等，比如击中角色后以哪里作为中心点显示击中的动画

## Public 属性

|                                                                         |
| ----------------------------------------------------------------------- |
| **id** : number;<br>ID                                                  |
| **x** : number;<br>坐标x                                                |
| **y** : number;<br>坐标y                                                |
| **width** : number;<br>宽度                                             |
| **height** : number;<br>高度                                            |
| **[type](#type)** : number;<br>类型：当前版本只支持矩形，未来将支持更多 |

## 详情

### type

```ts
type : number;
```

类型：当前版本只支持矩形，未来将支持更多

null/0=矩形

1=圆形

2=三角形

3=多边形

4=线段
