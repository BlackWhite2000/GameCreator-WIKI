## Public 属性

|                                                                                     |
| ----------------------------------------------------------------------------------- |
| id : number;<br>唯一ID                                                              |
| name : string;<br>名称                                                              |
| url : string;<br>图片路径 默认值=""                                                 |
| dataLayers : number[][][];<br>数据层数据 [自定义数据层索引][xGrid][yGrid] 默认值=[] |
| width : number;<br>宽度 默认值=0                                                    |
| height : number;<br>高度 默认值=0                                                   |

### getTileData

```ts [签名]

```

【静态】获取图块数据

- #### 参数

  - `texID` 图块素材ID

- #### 返回

  - `TileData` 图块素材配置数据
