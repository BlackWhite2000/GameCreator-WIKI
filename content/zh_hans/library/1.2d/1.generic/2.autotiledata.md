## 详情

### dataLayers

```ts
dataLayers : number[];
```

数据层数据-由于自动元件仅视为一格 默认值=[]
[dataGridIndex] = 1/0

### GCATMode

```ts
GCATMode : number;
```

规格模式

- 0-GCAT1的规格

### getAutoTileData

```ts
getAutoTileData(texID : number): AutoTileData
```

【静态】运行时使用，获取图块数据

#### 参数

- `texID` 自动元件的编号（1~N）
