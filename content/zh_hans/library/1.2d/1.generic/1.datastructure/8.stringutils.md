根据下方调用对应函数、属性即可

## Public 方法

|                                                                                                                                 |
| ------------------------------------------------------------------------------------------------------------------------------- |
| [getRealLength](#getreallength)(str : string): number<br>【静态】获取字符真实长度，会计算汉字                                   |

## 详情

### getRealLength

```ts
getRealLength(str : string): number
```

【静态】获取字符真实长度，会计算汉字

#### 参数

- `str` 字符串

#### 返回

- 字符串str的真实长度

### clearHtmlTag

```ts
clearHtmlTag(str : string): string
```

【静态】清除HTML格式

#### 参数

- `str` 可能带有HTML格式的字符串

#### 返回

- `[string]` 清除HTML格式后的字符串

### getMiddleDiff

```ts
getMiddleDiff(str1 : string, str2 : string): [number, number]
```

【静态】获取两个字符串中间不相同的地方的信息

#### 参数

- `str1` 字符串1
- `str2` 字符串2

#### 返回

- `[number, number]` \[头串相同的字符数目,尾串相同的字符数目\]
