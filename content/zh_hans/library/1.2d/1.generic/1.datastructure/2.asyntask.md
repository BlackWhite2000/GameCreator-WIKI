## 关联性

- `继承` 无
- `子类` 无

## 使用方式

多个任务同时进行，需要等待全部任务执行完毕后才回调

```ts
var task = new AsynTask(Callback.New(() => {<br>
  // 全部任务结束时的逻辑处理<br>
}, this));<br>
task.execute(任意表达式); // 如 task.execute(1);<br>
task.execute(任意表达式);<br>
task.complete(); // 上面执行了两个任务，下方则需要完成两次complete则算完成<br>
task.complete();
```
