## 属性说明

| 属性         | 属性变量名   | 类型   | 说明                                             |
| ------------ | ------------ | ------ | ------------------------------------------------ |
| 动画         | animationID  | number | 动画                                             |
| 水平方向缩放 | scaleNumberX | number | 横向缩放                                         |
| 垂直方向缩放 | scaleNumberY | number | 纵向缩放                                         |
| 播放模式     | playType     | number | 播放模式<br>0-不播放<br>1-播放一次<br>2-循环播放 |
| 帧率         | playFps      | number | 每秒播放的帧数                                   |
