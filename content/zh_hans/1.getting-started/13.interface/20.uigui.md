## 属性说明

| 属性     | 属性变量名        | 类型   | 说明                                                                                 |
| -------- | ----------------- | ------ | ------------------------------------------------------------------------------------ |
| 指定界面 | guiID             | number | 界面                                                                                 |
| 绑定类   | instanceClassName | string | 绑定类，系统会优先采用该处填写的绑定类，如果不存在再检测对应界面中预设的绑定类来使用 |
